# WIDAR: Real-time Wildlife Intrusion Detection & Response System
*(야생 동물 자동 검출 및 퇴치 시스템)*


WIDAR는 야생 동물이 민가/농가 주변에 출현하는 상황을 영상 기반으로 자동 탐지하고, 검출 결과에 따라 시각적/청각적 자극을 통해 야생 동물을 퇴치하는 시스템입니다. 야생 동물 영상에서 딥러닝 기반 객체 탐지 모델로 출현을 감지하며, 검출 시 퇴치 장치를 동작시켜 야생 동물의 접근을 방지합니다.
